# play-scala-snake-ladder

An extremely simple snake ladder game written in Play.

## Running

```bash
sbt run
```

### Start a game
```bash
curl -X POST -d '{"players":["Foo", "Bar"]}' -H "Content-Type: application/json" http://localhost:9000/games/start
```
